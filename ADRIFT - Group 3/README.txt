=== Opening the game ===

- Make sure that all 8 images and 4 audio files are together in the same folder as the "OPEN HERE" file.
- Execute the "OPEN HERE" file on your browser (there are some inconveniences with Chrome).

=== Extra Info ===

Version used: Twine 2.3.12 for Windows

Twine file: "Adrift Twine Game"
